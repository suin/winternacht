## Installation

1. Open the Extensions sidebar in VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for **Winternacht by suin**
3. Click **Install**
4. Open the theme picker (`Ctrl+K Ctrl+T` / `Cmd+K Cmd+T`)
5. Select **Winternacht Dark** or **Winternacht Light**
